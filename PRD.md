# Product Requirement Document (PRD)

## Project Title

WorkZen - Employee Management System

## Problem Statement

Many organizations still manage employee data, attendance,
leave requests, and payroll manually.

Manual systems are inefficient and prone to errors.

WorkZen provides a centralized web platform to manage employees,
track attendance, approve leave requests, and generate payroll reports.

## Objectives

- Digitize employee management
- Automate attendance tracking
- Simplify leave management
- Generate payroll and payslips
- Provide HR dashboard analytics

## Key Features

1. User authentication
2. Role-based access (HR and Employee)
3. Employee management
4. Attendance tracking
5. Leave management
6. Payroll processing
7. Payslip generation
